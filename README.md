# Сайт-резюме Сергея Котенко

Персональный сайт-портфолио frontend-разработчика. Создан на **Next.js 15** с использованием **React 19**, **TypeScript** и **styled-components**.

## 🚀 Технологии

| Категория       | Технологии                                                                 |
|-----------------|----------------------------------------------------------------------------|
| **Фреймворк**   | Next.js 15 (App Router)                                                    |
| **Язык**        | TypeScript 5.8                                                             |
| **UI-библиотека**| React 19                                                                   |
| **Стилизация**  | styled-components 6.1 + Framer Motion для анимаций                         |
| **Иконки**      | @styled-icons/feather                                                      |
| **Утилиты**     | classnames для условных классов                                            |

## 📁 Структура проекта

Проект использует архитектуру **Feature-Sliced Design (FSD)**:

```
src/
├── app/                    # Next.js App Router (маршрутизация и layout)
│   ├── layout.tsx          # Корневой layout с metadata и провайдерами
│   ├── page.tsx            # Главная страница (сборка всех виджетов)
│   ├── globals.css         # Глобальные стили
│   └── StyledComponentsRegistry.tsx  # Реестр styled-components для SSR
│
├── entities/               # Бизнес-сущности
│   └── candidate/          # Данные кандидата (тип, моки, API)
│
├── features/               # Функциональные возможности
│   ├── contact-info/       # Отображение контактной информации
│   └── skill-list/         # Список навыков
│
├── widgets/                # Композитные блоки (секции страницы)
│   ├── header/             # Шапка с навигацией
│   ├── hero/               # Главный экран с приветствием
│   ├── about/              # Секция "О себе"
│   ├── value/              # Секция с ценностями
│   ├── experience/         # Опыт работы
│   ├── skills/             # Навыки
│   ├── projects/           # Проекты
│   ├── achievements/       # Достижения
│   ├── contact/            # Контакты
│   └── footer/             # Подвал
│
├── shared/                 # Переиспользуемый код
│   ├── lib/                # Утилиты и хелперы
│   ├── styles/             # Глобальные стили, темы
│   ├── types/              # Общие типы TypeScript
│   └── ui/                 # Базовые UI-компоненты (кнопки, иконки)
│
└── pages/                  # Legacy-маршруты (если используются)
```

## 🧩 Описание компонентов

### Виджеты (widgets/)

| Компонент       | Описание                                                      |
|-----------------|---------------------------------------------------------------|
| **Header**      | Навигационная панель с именем и ссылками на секции            |
| **Hero**        | Главный экран: имя, должность, краткое описание, CTA-кнопки   |
| **About**       | Секция "О себе" с описанием опыта и специализации             |
| **Value**       | Блок с ключевыми ценностями в работе                          |
| **ExperienceWidget** | Хронология опыта работы (компании, должности, период)    |
| **SkillsWidget**| Список технических навыков сгруппированных по категориям      |
| **ProjectsWidget**| Галерея проектов с описанием и ссылками                     |
| **Achievements**| Достижения и сертификаты                                      |
| **Contact**     | Контактная информация с иконками (email, Telegram, GitHub)    |
| **Footer**      | Подвал с копирайтом и ссылками на соцсети                     |

### Фичи (features/)

| Компонент       | Описание                                                      |
|-----------------|---------------------------------------------------------------|
| **contact-info**| Компонент для отображения одного контакта (иконка + текст)    |
| **skill-list**  | Список навыков с прогресс-барами или тегами                   |

### Сущности (entities/)

| Сущность        | Описание                                                      |
|-----------------|---------------------------------------------------------------|
| **candidate**   | Типы данных кандидата и моки (имя, опыт, навыки, проекты)     |

## 🛠 Команды разработки

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера (localhost:3000)
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-сборки
npm start

# Линтинг кода
npm run lint
```

## 📦 Переиспользуемые утилиты (shared/)

- **lib/** — вспомогательные функции
- **styles/** — глобальные стили, CSS-переменные, миксины
- **types/** — общие TypeScript-типы
- **ui/** — базовые компоненты (Button, Icon, Container и т.д.)

## ⚙️ Конфигурация

- **next.config.ts** — оптимизация изображений (WebP/AVIF), сжатие, отключение `X-Powered-By`
- **tsconfig.json** — строгий TypeScript с путями (`@/` alias на `src/`)
- **StyledComponentsRegistry** — поддержка SSR для styled-components
